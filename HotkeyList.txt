Alt + 1 = 프로젝트 탐색기
Alt + 2 = 편집창 최대화
Alt + ` = 편집창 닫기 
Alt + F1 = 현재 파일 프로젝트 탐색기 창 위치 이동
Alt + j = 이전 탭 이동
Alt + k = 다음 탭 이동
Alt + l = 현재 파일 우측 창 분할 이동
Alt + h = 현재 파일 아래 창 분할 이동
Alt + - = 분할 창간 이동
Alt + Backspace = 분할 창 합치기
Alt + Shift + Backspace = 모든 편집창 닫기
Alt + F12 = 터미널창 이동
Alt + > = 터미널 새창 열기 
Ctrl + k = HotkeyList 이동
